HTML tables cheat sheet

Below is a table defining the key attributes and elements required to create a table using HTML

Element or attribute name Formatted tag Functionality of the element
Table <table> Contains all tabular data
Table Row <tr> Defines a new row
Table Heading <th> Defines a new heading
Scope scope = "col/row" Attribute determining if <th> is of row or column
Table Data <td> Contains all tabular data
Table Header <thead> Contains the column headings
Table Body <tbody> Contains all data excluding column headings
Table Footer <tfoot> Contains the bottom of the table, e.g. totals
Column span colspan = (#) Attribute determining how many columns the data will show on
Row Span rowspan = (#) Attribute determining how many rows the data will show on
